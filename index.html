Sports Timer V1.1

Description

Sports Timer is a application for Workout Timer / Training Timer for Firefox OS.
View Github

Features

  • Manage Session with exercises (add, delete, update)
  • Lock the screen during exercice
  • Pour chaque exercice vous définissez le temps de travail de repos et la répétition.
  • Alarm sound
  • Vibration

Interface

  1. Accés aux paramétrage (Ajout des séances et exercices.
  2. Liste des séances
  3. Liste des exercices pour la séance en court

User guide

Avant d'utiliser SportsTimer vous devez Créé une séance avec au moins un exercice.
