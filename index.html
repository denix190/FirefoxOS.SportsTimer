Sports Timer V1.1

Description

Sports Timer is a application for Workout Timer / Training Timer for Firefox OS.
View Github

Features

  • Manage Session with exercises (add, delete, update)
  • Lock the screen during exercice
  • For each exercice you define : Effort time, Recovery, and the numbers of rounds.
  • Alarm sound
  • Vibration

Interface

  1. Goto parameters (Add sessions and exercise).
  2. List of sessions
  3. List of exercises for the current session

User guide

Gives access to the settings of SportsTimer. Creation of sessions and exercises.
  • Parameters : Sound on/off
  • Session : Managing sessions.
  • About : About SportsTimer
